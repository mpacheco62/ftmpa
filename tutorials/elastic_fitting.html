Para ajustar la elasticidad se utiliza la ley de Hooke, desde la cual se obtienen los esfuerzos o deformaciones según las siguientes ecuaciones.

\[\boldsymbol{\epsilon} = \frac{1}{E}\left[(1+\nu)\boldsymbol{\sigma} - \nu\boldsymbol{I}\text{tr}(\boldsymbol\sigma)\right]\]
\[\boldsymbol{\sigma} = \frac{E}{1+\nu}\left[\boldsymbol{\epsilon} + \frac{\nu}{1-2\nu}\boldsymbol{I}\text{tr}(\boldsymbol\epsilon)\right]\]

One experiment¶

Para ajustar la ley de elasticidad se puede utilizar solo un ensayo, o multiples ensayos. El siguiente ejemplo demuestra como se ajusta el modelo elástico utilizando la librería lmfit y un solo ensayo.

elasticity_fit_1.py¶
 1 import lmfit
 2 import numpy as np
 3 import matplotlib.pyplot as plt
 4
 5 from ftmpa.experiments import ExpState, ExpData
 6 from ftmpa.models.handlers.elasticity import HandlerHooke
 7 from ftmpa.models.handlers.autoModels import AutoElasticity
 8 from ftmpa.for_lmfit import IterPrintForLmfit, ResidualVectorForLmfit
 9
10
11 # Making virtual experiment
12 strain_exp = np.linspace(0.0, 0.02, 100)
13 stress_exp = 100e3*strain_exp
14 stress_exp += np.random.normal(0, 20, len(strain_exp))
15
16 # Define the parameters of models
17 params = lmfit.Parameters()
18 params.add('younh', 1e3, min=0.0, max=2000e3)
19 params.add('nu', 0.3, vary=False)
20
21 # Define the name of parameters for models
22 h_elasticity = HandlerHooke(E="young", nu="nu")
23
24 # Define the auto calc
25 # this model calc the stress-strain curve for a stress 1e9 for default
26 model = AutoElasticity(h_elasticity=h_elasticity)
27
28 # Define the experiment and the function to obtain the results for the tensor stress and strain
29 exp_1 = ExpData(x=strain_exp, y=stress_exp,
30                 model_data={'x': lambda stress, strain: strain[...,0,0],
31                             'y': lambda stress, strain: stress[...,0,0]}
32                 )
33
34 # Define the experiment load status
35 exp_loads_1 = ExpState(model=model,
36                     results_test={'specimen1': exp_1},
37                     load_state=np.array([[1,0,0],[0,0,0],[0,0,0]], dtype=float))
38
39 # Minimize the function
40 out = lmfit.minimize(ResidualVectorForLmfit([exp_loads_1]), params, iter_cb=IterPrintForLmfit())
41 lmfit.report_fit(out.params)
42 final_params = out.params.valuesdict()
43
44
45 # Show the fitted parameters
46 data_fitted = exp_loads_1.data_to_graph(final_params)
47 plt.plot(data_fitted['specimen1']['x_exp'], data_fitted['specimen1']['y_exp'])
48 plt.plot(data_fitted['specimen1']['x_model'], data_fitted['specimen1']['y_model'])
49
50 plt.show()

2 import numpy as np
3 import matplotlib.pyplot as plt

En esta sección se importa la librería lmfit que nos permitirá ajustar los modelos, numpy para trabajar datos y matplotlib para graficarlos.

5 from ftmpa.experiments import ExpState, ExpData
6 from ftmpa.models.handlers.elasticity import HandlerHooke
7 from ftmpa.models.handlers.autoModels import AutoElasticity
8 from ftmpa.for_lmfit import IterPrintForLmfit, ResidualVectorForLmfit

Se importan las librerías propias de esta documentación, en esta parte no se daran detalles de ellas, pero sí mas adelante cuando se utilizen.

5 # Making virtual experiment
6 strain_exp = np.linspace(0.0, 0.02, 100)
7 stress_exp = 100e3*strain_exp
8 stress_exp += np.random.normal(0, 20, len(strain_exp))

Se crean datos virtuales para un ensayo de tracción, además se le agrega ruido.

16 # Define the parameters of models
17 params = lmfit.Parameters()
18 params.add('young', 1e3, min=0.0, max=2000e3)
19 params.add('nu', 0.3, vary=False)

En estas líneas se definen TODOS los parametros que van a necesitar los modelos, por medio de parámetros de lmfit. Aca se definene los valores iniciales de los parámetros, cuales varían y en que rangos.

21 # Define the name of parameters for models
22 h_elasticity = HandlerHooke(E="young", nu="nu")

Se define que el manejador será HandleHooke, que el parámetro del modelo de elasticidad “E” se pasara con nombre de “young”, y que el parámetro “nu” se pasara con nombre “nu”.

24 # Define the auto calc
25 # this model calc the stress-strain curve for a stress 1e9 for default
26 model = AutoElasticity(h_elasticity=h_elasticity)

Se define el modelo que va a automaticamente crear “simulaciones” para el caso en estudio.

28 # Define the experiment and the function to obtain the results for the tensor stress and strain
29 exp_1 = ExpData(x=strain_exp, y=stress_exp,
30                 model_data={'x': lambda stress, strain: strain[...,0,0],
31                             'y': lambda stress, strain: stress[...,0,0]}
32                 )

Aquí se definen los experimentos y como se obtienen sus resultados respecto del tensor \(\boldsymbol \sigma\) (stress) y \(\boldsymbol \epsilon\) (strain).

En este caso se construyo la curva como deformación en el eje “x” (línea 6) y como esfuerzo en el eje “y” (linea 8). Por lo tanto se necesita que el dato “x” sea la deformacion \(\epsilon_{xx}\) (strain[...,0,0]) y que el dato “y” sea el esfuerzo \(\sigma_{xx}\) (stress[...,0,0]).

34 # Define the experiment load status
35 exp_loads_1 = ExpState(model=model,
36                     results_test={'specimen1': exp_1},
37                     load_state=np.array([[1,0,0],[0,0,0],[0,0,0]], dtype=float))

Aca es donde se le define al programa que estado de tensión se está aplicando y que modelo lo debe representar. El parámetro model representa el modelo que se definió en la línea 26 y que es el AutoElasticity.

El parámetro result_test es un diccionario donde se deben colocar todos los ensayos experimentales que tengan el mismo estado de tensión, en este caso una tracción simple.

En load_state se debe definir cual es el estado de tensiones que se está imponinendo en el ensayo. Para este caso como es un ensayo de tracción simple, corresponde a una matriz que solo tiene esfuerzos en \(\sigma_{xx}\). Esto se representa por el código np.array([[1,0,0],[0,0,0],[0,0,0]], dtype=float)

Con esto queda todo definido para las funciones de la librería lo siguiente es para realizar la minimización del residuo.

39 # Minimize the function
40 out = lmfit.minimize(ResidualVectorForLmfit([exp_loads_1]), params, iter_cb=IterPrintForLmfit())
41 lmfit.report_fit(out.params)
42 final_params = out.params.valuesdict()

En este codigo se utiliza lmfit para realizar la minimización, esta librería ya trae incorporadas unas funciones de residuos y de iteraciones que son opcionales ResidualVectorForLmfit([...]) y IterPrintForLmfit(). En la inicialización de ResidualVectorForLmfit([...]) debe introducir todos los estados de tensiones que se deban ajustar.

Luego que la minimización se haya realizado (línea 40), se reporta por la salida estandar el reporte final del ajuste mendiante la línea 41.

Finalmente se guardan los parámetros finales, como diccionario, del ajuste en la variable final_params.

45 # Show the fitted parameters
46 data_fitted = exp_loads_1.data_to_graph(final_params)
47 plt.plot(data_fitted['specimen1']['x_exp'], data_fitted['specimen1']['y_exp'])
48 plt.plot(data_fitted['specimen1']['x_model'], data_fitted['specimen1']['y_model'])
49
50 plt.show()

En estas líneas se obtienen la curvas de los ajustes realizados y se grafican, la línea 46 realiza los calculos con lo parámetros finales. Para cada estado de carga se pueden obtener sus ajustes, en este caso solo contamos con exp_loads_1 y que tiene solamente un ensayo 'specimen1'. Por lo tanto para obtener estos valores debemo primero debemos indicarlo como data_fitted['specimen1'], dentro de este diccionario tenemos cuatro datos x_exp los datos experimentales “x”, y_exp los datos experimentales “y”, x_model los datos “x” del modelo y y_model los datos “y” del modelo.

Con estos datos se grafica el ajuste realizado.
